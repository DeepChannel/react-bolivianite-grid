# Copy and paste

Example of grid, based on [this](/examples/editable) example, that implements copy/paste feature.

Notes:
- `CMD/CTRL`+`C` - copy cells
- `CMD/CTRL`+`OPT/ALT`+`C` - copy cells with headers
